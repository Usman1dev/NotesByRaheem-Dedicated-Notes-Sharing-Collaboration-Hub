# 📚 NotesByRaheem.xyz
**The Dedicated Notes-Sharing & Collaboration Hub**

NotesByRaheem.xyz is a streamlined academic platform built to centralize resource sharing and foster real-time collaboration. Developed by **Usman** and **Abdur Raheem**, the system ensures high-quality study materials through administrative oversight and provides a modern space for class-wide discussion.

---

## ✨ Key Features
- **Verified Notes:** Secure upload/download system with Admin-led quality control.
- **Real-Time Chat:** Dedicated course-specific chatrooms and a global discussion board.
- **Leaderboard:** Recognition and ranking for the top academic contributors.
- **Secure Access:** Manual account provisioning to ensure a spam-free environment.

---

## 💻 Tech Stack
- **Frontend:** React 18, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (Auth, PostgreSQL, Storage, Edge Functions)

---

## 🚀 Quick Start
1. **Request Access:** DM the developers with your Name, Roll No, and Username.
2. **Login:** Visit [notesbyraheem.xyz](https://notesbyraheem.xyz).
3. **Contribute:** Upload your notes and start earning your spot on the leaderboard!

---

## ⚠️ Community Guidelines
- **Quality Only:** Ensure all uploads are legible and relevant.
- **Respect:** Zero tolerance for harassment or off-topic spam.
- **Integrity:** Adhere to university academic honesty policies.
